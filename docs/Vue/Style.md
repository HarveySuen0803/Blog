# Style Label

```html
<style lang="scss">
.box {
  font-size: 50px;
  p {
    color: pink;
  }
}
</style>
```

# SCSS Style

Install Sass and Sass Loader.

```shell
npm install -D sass
npm install -D sass-loader
```

Use SCSS in your Vue files.

```vue
<style lang="scss">
.box {
  font-size: 50px;
  p {
    color: pink;
  }
}
</style>
```
